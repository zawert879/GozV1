# Game of Zawert - Руководство по использованию с ИИ-ассистентом

Добро пожаловать в руководство по использованию фреймворка "Game of Zawert" (GOZ) с ИИ-ассистентами, такими как GitHub Copilot. Этот документ поможет вам быстро и эффективно генерировать контент для ваших игр.

## 1. Как правильно давать контекст

Чтобы ИИ-ассистент мог генерировать персонажей и предметы в соответствии с правилами, ему необходимо сначала "прочитать" эти правила.

**Ваш главный инструмент — папка `GOZ_Framework`.**

Перед тем как дать команду на генерацию, всегда начинайте свой запрос с указания ассистенту использовать эту папку.

#### Пример базовой инструкции:

Вы можете просто скопировать и вставить это в чат с Copilot, а затем дописать свой запрос.

> Используя файлы из папки `#file:GOZ_Framework`, [ВАШ ЗАПРОС]

---

## 2. Примеры запросов (Промптов)

Вот несколько примеров, которые вы можете адаптировать под свои нужды.

### Генерация Персонажа

**Основная команда:** `Создай мне персонажа по концепции "[ОПИСАНИЕ]"`.

*   **Простой запрос:**
    > Используя файлы из папки `#file:GOZ_Framework`, создай мне персонажа по концепции "сильный, но глупый варвар".

*   **Более детальный запрос:**
    > Используя файлы из папки `#file:GOZ_Framework`, сгенерируй персонажа: "старый, мудрый, но физически слабый волшебник, специализирующийся на защитных заклинаниях".

*   **Запрос для другого сеттинга:**
    > Используя файлы из папки `#file:GOZ_Framework`, создай персонажа для мира киберпанк: "уличный самурай с кибер-руками, быстрый и смертоносный, но не очень общительный".

### Генерация Снаряжения

**Основная команда:** `Создай таблицу из [X] предметов для сеттинга "[ОПИСАНИЕ СЕТТИНГА]"`.

*   **Оружие:**
    > Используя файлы из папки `#file:GOZ_Framework`, создай таблицу из 5 видов энергетического оружия для сеттинга "космический вестерн".

*   **Магические предметы:**
    > Используя файлы из папки `#file:GOZ_Framework`, сгенерируй 3 магических свитка и 3 алхимических зелья для классического фэнтези.

*   **Предметы для другого сеттинга:**
    > Используя файлы из папки `#file:GOZ_Framework`, мне нужен список из 4 кибернетических имплантов для мира киберпанк, с их эффектами и ценой.

## 3. Лучшие практики: "Живой" список предметов

Файл `Sample_Adventure.md` — это не просто пример, а ваша постоянно растущая база данных предметов. Чем больше в ней будет примеров, тем точнее ИИ сможет балансировать новые предметы в будущем.

### Рекомендуемый рабочий процесс:

1.  **Сгенерируйте новые предметы**, используя промты из раздела выше.
2.  **Проверьте результат.** Убедитесь, что предметы вам нравятся и соответствуют духу вашего мира.
3.  **Дайте команду на обновление.** Попросите ассистента добавить новые, одобренные вами предметы в `Sample_Adventure.md`.

#### Пример промта для обновления:

> Отлично, мне нравятся эти предметы. Добавь их в файл `#file:Sample_Adventure.md` в соответствующий раздел.

Этот простой шаг со временем значительно улучшит качество генерации и сделает вашу личную базу снаряжения богаче и разнообразнее.
